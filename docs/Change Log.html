Change log of Wolf Engine

  • New features
  • Bugs fixed
  • Performance improved

Last updated: 3/25/2016_3:37 AM

  • v 0.1.0.0
    • Added game class for wolf
    • Wolf.ffmpeg added, you can handle video and audio with this library
    • Shapes (ellipse, line, simple rounded rectangle and geometry) have been added to "Wolf::Graphics::Direct2D::Shapes" namespace
    • DirectX 11.X renderer framework implemented under wolf::graphics namespace
  • v 0.2.0.0
    • lightweight "GUI framework" added for DirectX 11.X.(includes label, button (with icon), radioButton, checkBox, scrollBar, comboBox, slider, listBox(with icon), image
    • Integrated with "Intel TBB build tbb44_20160526oss"
    • "Blend states" have been added
    • function "w_time_span::from_string" has been added
    • Integrated with "ffmpeg build "git-78baa45 (2016-04-28)""
    • Integrated with "Python 3.5.1 ". Now you can execute python files directly using "wolf::system::w_python" class.(Python 3.5.1 x64 must be installed on your target machine)
    • Load basic geometries from "collada files".
    • "Async/Sync" send and received TCP client implemented.
    • "Graphics diagnostic debugging" implemented. Also Wolf prevents from debugging the release application.
    • Used "localtime_s instead of deprectaed localtime" in w_io.h
    • Updated to "Blackmagic DeckLink SDK 10.6.4"
